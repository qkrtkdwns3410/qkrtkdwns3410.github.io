## 안녕하세요!

새롭게 리뉴얼한 블로그에 오신 것을 환영합니다.

이 블로그는 **Next.js 14**와 **Tailwind CSS**로 만들어졌으며, GitHub Pages에 배포되어 있습니다.

### 주요 기능

* 깔끔한 미니멀 디자인

* 다크 모드 지원

* 노션 스타일 에디터로 글 작성

* 시리즈별 글 관리

* 태그 시스템

* 반응형 디자인

### 글 작성 방법

1. `/write` 페이지에서 노션 스타일 에디터로 글을 작성합니다

2. MDX 파일로 다운로드합니다

3. `src/content/posts/` 디렉토리에 파일을 넣습니다

4. git push하면 자동으로 배포됩니다
